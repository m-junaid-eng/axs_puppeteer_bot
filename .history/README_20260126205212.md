# Puppeteer opener (compliant)

This project opens a URL using Puppeteer and saves a screenshot.

## Install

```bash
npm install
```

## Run

```bash
npm run open:axs
```

### Options

- Open a custom URL:
  ```bash
  npm run open:axs -- --url "https://example.com"
  ```

- Use a specific User-Agent (for compatibility testing):
  ```bash
  npm run open:axs -- --ua "Mozilla/5.0 (...)"
  ```

- Rotate UAs from `uas.json` (stores used UAs in `.state/used_uas.json`):
  ```bash
  npm run open:axs
  ```

- Run headless:
  ```bash
  npm run open:axs -- --headless
  ```

- Save the loaded page HTML (default on), or disable it:
  ```bash
  npm run open:axs -- --no-save-html
  ```

- Follow "All Events" then open "See Event" links and save each event HTML:
  ```bash
  npm run open:axs -- --all-events --max-events 10 --delay-ms 1500
  ```

- Interactive mode (for flows that show a CAPTCHA on event pages):
  ```bash
  npm run open:axs -- --all-events --interactive --max-events 10
  ```
  When a CAPTCHA page appears, solve it in the browser window, then press Enter in the terminal to continue.

- Export event details to Excel (XLSX) (default on when using `--all-events`):
  ```bash
  npm run open:axs -- --all-events --xlsx-path "artifacts/events.xlsx"
  ```
  Disable with `--no-xlsx`.

- Use a fresh browser profile each run (no cookies/cache carryover):
  ```bash
  npm run open:axs -- --fresh-profile
  ```

- Save fingerprint *signals* for learning/debugging (default on), or disable it:
  ```bash
  npm run open:axs -- --no-fingerprint
  ```

## Outputs

- Latest screenshot: `artifacts/axs.png`
- Latest HTML: `artifacts/axs.html`
- Latest fingerprint signals (JSON): `artifacts/fingerprint.json`
- All Events HTML (when using `--all-events`): `artifacts/all-events.html`
- Event HTML/screenshot (when using `--all-events`): `artifacts/events/*.html` and `artifacts/events/*.png`
- Event details JSON (when using `--all-events`): `artifacts/events.json`
- Event details XLSX (when using `--all-events`): `artifacts/events.xlsx`
- Fingerprint history (JSON per run): `artifacts/fingerprints/<timestamp>.json`
- Fingerprint text log (TSV): `artifacts/fingerprints/fingerprints.tsv`
